    CASTOR

    Log-file encryption and authentication utility toying with Attribute Based Encryption (ABE) and Identity Based Authentication (IBA), along AES and RSA.

    Castor serves as a syslog message processer and is able to handle any message respecting the syslog format, while adaptations to any other format are simple.

    Install & Compile

    Get the code:

    git clone git://github.com/Shastick/castor.git

    cd castor

    Initialize and update the submodules (to pull the cpabe library)

    git submodule init

    git submodule update

    If there is anything else than the two cpabe and bswabe directories in cpabe/src/main/java/, move to the specified directory and switch to the stripped branch:

    cd cpabe/src/main/java/

    git checkout stripped

    You should now be ready to compile:

    sbt compile

    And if you want to run castor from sbt, you'll have to move the conf and file directories to the castor root.

    mv castor/conf ./

    mv castor/files ./

    sbt compile

    sbt run

    Configuration

    Look at https://github.com/Shastick/castor/blob/master/castor/conf/digester_config.example.scala and https://github.com/Shastick/castor/blob/master/castor/conf/reader_config.example.scala for configuration examples.

    Have fun, and don't hesitate if you have any questions! J.
